Figure 15. Stock Calculation Logic

Function: getItemStock(itemId) — transaction-based inventory (no static stock field)

Start Input: itemId
Initialize stock = 0
Query database GROUP BY type on transactions WHERE itemId = ? SUM(quantity) per type (IN, OUT, RETURN)

For each transaction type group:

IN Receive stock
stock += qty
RETURN Returned by borrower
stock += qty
OUT Issued to borrower
stock −= qty
Floor at zero Prevent negative display return max(0, stock)
Output currentStock (used on dashboard, inventory, scan)
IN — adds to stock RETURN — adds to stock OUT — subtracts from stock

Pseudocode (server/items.ts)

function getItemStock(itemId):
    groups = db.transaction.groupBy(type, itemId)
    stock = 0

    for each row in groups:
        qty = row.sum(quantity) ?? 0
        if row.type is "IN" or "RETURN":
            stock += qty
        else if row.type is "OUT":
            stock -= qty

    return max(0, stock)
